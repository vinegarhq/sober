Introducing Sober

We're testing Sober, a brand new way to get Roblox on your Linux desktop. It's a novel approach, separate to other compatibility software like Wine, Android emulators, or virtual machines. Sober works by creating a specialized runtime just for Roblox's code, ensuring compatibility.

We'd like to stress that Sober is an experimental project. It's a passion project rather than production-grade code. Sober's runtime incurs a performance penalty compared to Windows. While the results vary for everyone, generally Sober offers higher performance compared to Windows.

Sober is unfortunately closed-source. This could change in the future.

We distribute Sober as a Flatpak package. Here's how to use it:

  1. Make sure Flatpak is installed
  2. Install the package using one of the methods below
  3. Open your favorite Roblox experience
  4. Sober will prompt you for installation
  5. You now have full access to Roblox!

Sober is a labor of love, and we'd love for you to try it out! We hope you enjoy your time.
